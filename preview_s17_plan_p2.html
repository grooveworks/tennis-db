設計メモ (実装前の自分用)

Strategy 圧縮の数値: padding 10px 12px → 6px 4px、bullet 6px → 5px、行アイコン 26px → 24px (opacity 0.32、hover 0.85)、background panel2 → 透明 (点線 divider のみ)。1 項目高さ ~50px → ~32px (約 36% 圧縮)。

Gear Decision の冒頭表示: Phone A で Header 44 + Target 約 195 + Strategy 約 215 + 余白 = 約 460px、phone body 632px なので Gear 見出し (約 42px) は 1 画面内に収まる計算。

編集モード切替の検討 (今回 p2 では実装せず): ChatGPT 案では「編集モード時にアイコンを強調」とあった。今は常時 opacity 0.32 で hover 時 0.85、十分控えめなのでモード切替なしで様子見。 PC では hover 効くがスマホでは hover 効かないため、スマホで実機検証時に「アイコン見えにくい」となれば編集モード切替を S17.x で追加検討。

Target Edit Modal は変更なし: 大会選択の「練習イベント」追加は S17 後 (S17.1 Practice Focus Card 着手時) に検討。

「大会変更」と「編集」の役割分担: 「大会変更」= ターゲットの大会自体を別の大会に差し替え、「編集」= 目標・テーマだけ変更。意味の違いを明示。
