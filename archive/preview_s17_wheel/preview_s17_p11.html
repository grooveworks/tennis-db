v11 で適用した CSS (静的でも 3D 表現可):
・mask-image: linear-gradient(to bottom, transparent 0%, #000 22%, #000 78%, transparent 100%) = 上下端フェード
・perspective: 600px + transform: rotateX(NNdeg) scale(0.X) = 円筒に貼り付いた錯覚
・中央帯は ::before / ::after で 1px ライン 2 本のみ (塗りつぶし無し)
