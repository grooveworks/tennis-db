FINAL Home の構成 (5 カード + Quick Add)
  • Header: TennisDB + version + 同期 + 天気 (タップで詳細展開、tinted bg で示唆) + ユーザー
  • Quick Add: 大会 / 練習 / 試打 の 3 大ボタン (絶対変えない)
  • 1. 現在の状況 (Current Context): 次の大会 / 課題 / 主力 [継続使用中] / 検討中 [採用候補] / 直近結果 — Gear Context をここに統合
  • 2. 今週サマリー: 直近 7 日 — 練習 3 / 大会 1 / 平均 80 分/回 / 直近 7 勝 3 敗 (大数字 34px tabular)
  • 3. 次のアクション: 上位 3 件 (priority dot + チェック circle、タップで完了化は S17 で本格)
  • 4. カレンダー: 4/27〜5/10 の 2 週間、日付タップで DayPanel 展開
Glass の使用範囲: DayPanel と Modal のみ。通常 Home カードには適用しない (情報密度確保のため)。
角丸ヒエラルキー: Card 20 / 行カード (DayPanel 内) 14 / ボタン 12 / バッジ 6-8。
hover lift: PC/タブレット (`@media (hover:hover) and (pointer:fine)`) のみ、スマホでは無効。
