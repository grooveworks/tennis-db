設計メモ (実装前の自分用)

カード優先順位 (上から): ① 次のターゲット → ② 今回の作戦 → ③ 今回のギア決定。 Target が起点、Strategy と Gear は Target に紐付く。

編集ボタン規律: 各カード右上に [編集] 1 個。Target だけ [変更] も追加 (大会の差し替え用)。Strategy は行ごとに小さい編集/削除アイコンで個別操作。

Target Edit Modal: 大会選択は tournaments[] の日付未来順。過去大会も区切り線で表示 (振り返り目的の利用想定)。auto-save 対応。

Strategy Card: 最大 5 項目、5 項目達後は + ボタン非表示。並び順は追加順固定 (S17 では Reorder Mode 適用なし、5 項目以下なので不要)。

Gear Decision Card: 本命 (橙アクセント) / 対抗 (ミントアクセント) / 保留 (グレー) の 3 段。本命のみ必須、対抗・保留は任意。空段はラベルだけ表示。全体懸念は黄色帯で警告色。

ターゲット未設定時: Strategy / Gear カードは opacity 0.45 + ボタン無効、上に青い案内 (callout) で「先にターゲットを設定」と誘導。

役割分離: Plan は意思決定、Sessions は過去事実、Gear は道具情報、Home は直近予定。詳細比較は Gear タブ、過去結果は Sessions タブで見る。

S17 後の追加予定 (今回は実装しない): Practice Focus Card / Pre-match Checklist / Opponent DB (50 名超対戦相手管理は別 Stage)。
