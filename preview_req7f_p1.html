仕様
UI 動作:
• 「並び替え」リンク → reorderMode=true → ▲▼ 表示、ヒント帯表示、「完了」ボタン
• ▲▼ で順序変更 → 即時反映 (Strings と同じ visible[i] と visible[i±1] の order swap)
• 「完了」で reorderMode=false → 元の caret-right 表示に戻る
• 並び替え中は status filter (主力/サブ/候補/引退) は維持 (ただし内部ロジックは status priority → order の二段ソート)
• 「並び替え」リンクは追加ボタンの左側、Strings と並び順位置を揃える

影響範囲:
• src/ui/gear/GearTab.jsx Racket Board セクション (行 370-414 あたり) のみ変更
• src/ui/gear/StringsSection.jsx から ReorderToggleLink, ReorderDoneButton, ReorderHintBar, swapWithVisible パターンを再利用 (新規 helper 不要)
• app.jsx persistRackets は既存のまま、引数 newList でそのまま動く
• 機能・データ層変更なし、純粋に UI 追加
