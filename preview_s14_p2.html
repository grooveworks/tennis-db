preview_s14_p2 — 天気詳細 Modal (Glass、ボトムシート)

P2 範囲: Header の天気スロット (22° 橙 tinted) タップで Glass モーダルが下からせり上がる。Open-Meteo API を hourly 拡張 (key 不要)。
UI 構成: グリップ → ヘッダ「今日の天気 4/27(月)」 + 閉じる × → メインブロック (現在気温 40px 大数字 + アイコン + 状態 + 最高/最低) → 時間別 (横スクロール 16 セル、現在時刻 17:00 を Apple Blue ハイライト) → フッタ注釈 (位置 + Open-Meteo)。
S20 スコープ外: 大会・練習との関連表示 (4/29 大会の予報など) は S20 (自動連関) で実装。P2 では純粋な天気詳細のみ。

P2 確認チェックリスト
  • ① Header 天気タップでハイライト: 22° セルが橙 tinted bg のさらに濃いめ + 外周薄リングで「タップ中」を示す (Modal 開いている間)
  • ② Glass モーダル: 画面下から TabBar (60px) のすぐ上に bottom-sheet で出現、上端のみ角丸 22px、グリップ 36×4px (引きおろし可ヒント)
  • ③ メインブロック: 左に天気アイコン 54px (晴れ=Phosphor fill sun、橙)、右に気温 40px Display tier「22°」+ 状態テキスト「晴れ」+ 「最高 24° / 最低 15°」
  • ④ 時間別: 横スクロール 16 セル (06:00 - 21:00)。各セル 56px 幅で「時刻 / アイコン / 気温」。現在時刻のセル (例 17:00) を Apple Blue tinted bg + 外周リング + 「17 今」ラベルでハイライト
  • ⑤ アイコン使い分け: 晴れ=sun (橙 fill) / 晴れ時々曇=cloud-sun (グレー regular) / 曇=cloud (グレー) / 雨=cloud-rain / 雪=cloud-snow / 雷=cloud-lightning
  • ⑥ フッタ注釈: 「位置: 埼玉県 (35.85, 139.65) · データ: Open-Meteo」を 10px グレーで控えめに
  • ⑦ 閉じる導線: 右上 × ボタン / 背景 dim タップ / グリップ下スワイプ (実装簡略のため Esc + 背景タップで閉じる予定、グリップは視覚的ヒントのみ)
S14 P2 で「やらない」こと (S20 以降に送る):
  • 大会日との関連表示 (例: 4/29 大会の予報、雨注意)
  • 週間予報 (今日のみ。明日以降を見たければ Sessions タブのカレンダー側で示唆)
  • UV指数 / 風速 / 湿度 (情報過多回避)
