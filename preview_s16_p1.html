判断してほしいこと

1. ラベル名 (status 6 種) — 提案: 主力 / 予備 / 候補 / 検討中 / 用途別 / 引退。実機材を当てはめた感覚で違和感あれば言葉を直してください (用途別 → 練習用 / フォーム機 / 特化機 / 補助機 候補)。
2. Decision Hub の構成順序 — Current Setup → Racket Board → Recent Trials → Open Questions → Manage Masters。逆にしたい / 削りたいセクションあれば指示ください。
3. Racket Detail の優先順位 — Summary → Current Setting → Trial Summary → Usage → Decision Notes → Measurements。「実測値が下すぎる」「Decision Notes をもっと上に」等の感覚があれば修正します。
4. Open Questions の置き場 — Gear タブに置く案ですが、S17 Plan タブとの被りが気になる場合は Plan に集約する選択肢もあります。Gear に置くべきか、置かないか。
5. Manage Masters の階層 — トップに 1 タップ折り畳み 3 行で配置。「Strings 在庫はトップに展開して見せたい」等あれば指示を。
